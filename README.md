# eda-lab-7

Бот №1 — Офіціант (bot1_waiter.py)

connect_rabbitmq():
Намагається підключитись до RabbitMQ
Якщо не вдалось — чекає 3 секунди і пробує знову
Повторює поки не підключиться
Потрібна бо при старті Docker бот може запуститись раніше ніж RabbitMQ

send_to_queue():
Отримує словник із замовленням
Кожного разу створює нове з'єднання (щоб уникнути Channel is closed)
queue_declare — створює чергу якщо не існує, або підключається до існуючої
basic_publish — відправляє повідомлення в чергу
json.dumps — перетворює словник у рядок
Закриває з'єднання одразу після відправки

start()
Спрацьовує коли користувач пише /start

handle_buttons()
Обробляє всі натискання кнопок
